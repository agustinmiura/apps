Layout

* flex : A layout with flexbox
    Supports 3 resolutions : 
    * Desktop : From a width of 1334 px .
    * Tablet : Up to a width of 1334 px.
    * Mobile : Up to a width of 646 px.

 * grid-layout : Layout with grid css.
